Hoe upload ik audioclips?

Ga naar je plugins folder > SheepAudio > audioclips

Daar kan je .mp3 bestanden neerzetten.

JE MOET EEN ambient.mp3 HEBBEN ANDERS START DE PLUGIN NIET OP

dit is de audio die je hoort als je niet in een region bent (ik raad vogels aan)

 

Hoe gebruik ik de regions?

doe //wand om een wand te krijgen. Selecteer (linker en rechter muisknop) de ene hoek van je region en dan de andere.

doe dan /rg create [naam van audiobestant] eg. /rg create vogels

 

Hoe kan ik meerdere regions de zelfde audio geven?

Zoals je waarschijnlijk hebt gemerkt kan je niet 2 regions de zelfde naam geven. Daar hebben we een oplossing voor! Voeg een "+' achter je naam. eg.

1ste region: "/rg create koeien"

2de region: "/rg create koeien+"

3de region: "/rg create koeien++"
